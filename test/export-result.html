附录：计算公式说明

运营收入合计 = 门诊计奖收入 + 医事服务费（不含药耗）+ 病房计奖收入 + 医事服务费

运营支出合计 = 人员支出 + 设备支出 + 非收费医材支出 + 其他消耗支出 + 第三方服务支出

其他消耗支出 = 试剂 + 设备类耗材 + 办公耗材 + 医疗废弃物 + 氧气

设备支出合计 = 设备折旧 + 设备维修

支收比 = 科室支出合计 / 科室计奖收入合计

注意事项指标 = （人员经费 + 科室支出）/ 科室计奖收入
