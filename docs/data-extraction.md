## 一、科室运营指标表（图片1）

**标题**：2026年2月病理科运营指标

**表格结构**：行 = 指标名，列 = 时间维度

| 列 | 含义 |
|:---|:---|
| A | 运营指标\提供数据周期 |
| B | 2025年度均值 |
| C | 本年度均值 |
| D-N | 1月情况 ~ 11月情况 |

**指标行清单**：

### 一、科室基本信息
- 科室人数：44 (2025均值) → 47 (本年度/1月/2月)

### 二、科室经济指标

**(一) 门诊收入**
- 门诊计奖收入总计
- 门诊医事服务费费
- **门诊运营收入总计**（加粗行，汇总项）
- 门诊流水收入总计
- 门诊材料费
- 门诊药费

**(二) 科室支出**
1. 售类消耗
   - 非收费耗材
   - 办公耗材
   - 试剂领用
   - 医疗垃圾
   - 氧气
   - 设备类耗材
2. 第三方费用
3. 人员工资
4. 兼职教授成本
5. 设备折旧（以附件形式提供明目）
6. 设备维修费
7. 基本奖金
8. 夜班费

**(三) 科室运行情况汇总**
- 科室计奖收入合计
- 科室支出合计

### 三、科室工作量指标

**(一) 门诊工作量情况**
- 专家门诊量：0, 0, 0, 0（病理科无专家门诊）
- 特需门诊量：0, 0, 0, 0（病理科无特需门诊）
- 普通门诊量：575, 569, 663, 475

**科室需注意事项**：（人员经费+科室支出）/科室计奖收入

### 四、备注（计算公式）

- 运营收入合计 = 门诊计奖收入 + 医事服务费（不含药耗，其他收入按30%/70%计入开单/执行科室）+ 病房计奖收入 + 医事服务费（不含药耗，检查化验收入按30%，手术费及麻醉50%，其他收入100%计入科室）
- 运营支出合计 = 人员支出 + 设备支出 + 非收费医材支出 + 其他消耗支出 + 第三方服务支出
- 其他消耗支出 = 试剂 + 设备类耗材 + 办公耗材 + 医疗废弃物 + 氧气
- 设备支出合计 = 设备折旧 + 设备维修
- 设备折旧（资金来源自筹）= 设备上账金额/60，按月扣除；六到十年逐年递减
- 设备维修（资金来源自筹及财政）= 设备上账金额×0.005，按月扣除
